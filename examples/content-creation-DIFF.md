## SHADOW SCAN

**6 instruction shadows found:**

1. **Blog Introduction** — TEMPLATE_INSTRUCTION — "hook the reader with a question, statistic, bold claim, or relatable scenario" → presents a menu of opening templates instead of guiding toward the strongest entry point for THIS topic. Produces S-03 Template Cascade.

2. **Blog Conclusion** — CLOSURE_INSTRUCTION — "summarize key takeaways, reinforce the main message, include a call to action" → every blog post ends with the same three-part closure. Produces S-07 Closure Rush.

3. **Blog Body** — VOLUME_INSTRUCTION — "3-5 sections" as default → forces padding when the topic needs fewer sections. Produces S-02 Helpful Explosion through structure.

4. **Hook Formulas** — BALANCE_INSTRUCTION — 6 hook formulas presented as equal options with no guidance on when to use which. Produces S-06 False Balance — the AI picks one at random or lists them all.

5. **Channel Tone** — WARMTH_INSTRUCTION — "conversational tone" (Facebook), "professional but human" (LinkedIn) → generic tone guidance that produces predicted warmth across all posts. Produces S-08 Sophisticated Authenticity.

6. **CTA Section** — EXPLOSION_INSTRUCTION — 5 CTA examples per context type → AI reproduces volume instead of picking the one that fits. Produces S-02 Helpful Explosion.
